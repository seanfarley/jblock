
* jblock - j's adblocker

~jblock~ is a WIP adblocker. We'll see if it ever gets finished.

* Installation
** Dependencies
- ~python3~ or ~pypy3~
- ~attrs~
** Tests
- ~pytest-benchmark~

If you pass pep8, you fail the style tests.
